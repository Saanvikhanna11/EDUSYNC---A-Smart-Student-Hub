# 🎓 EDUSYNC — A Smart Student Hub

## 📌 Overview

**EduSync** is a Java-based student management system designed to help students efficiently manage their academic activities in one place. It provides features like GPA tracking, attendance monitoring, task management, reminders, and subject-wise performance analysis.

---

## 🚀 Features

* 📊 **GPA Calculator** – Track and calculate academic performance
* 📅 **Attendance Tracker** – Monitor attendance percentage
* 📝 **Task Manager** – Add and view tasks with deadlines
* ⏰ **Reminders** – Keep track of important events and deadlines
* 📚 **Subject Management** – Store marks and grades for each subject
* 📈 **Performance Report** – View complete academic report

---

## 📂 Project Structure

```
EduSync/
│── models/        # Data classes (Student, Subject, Task, etc.)
│── services/      # Business logic (GPA, Attendance, Auth, etc.)
│── main/          # Main application entry point
│── interfaces/    # Interfaces used in the project
│── exceptions/    # Custom exception handling
│── users.txt      # Sample user data
```

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/Saanvikhanna11/EDUSYNC---A-Smart-Student-Hub.git
```

2. Navigate to the project folder:

```
cd EDUSYNC---A-Smart-Student-Hub
```

3. Compile the project:

```
javac models/*.java services/*.java main/*.java interfaces/*.java exceptions/*.java
```

4. Run the application:

```
java main.mainApp
```
